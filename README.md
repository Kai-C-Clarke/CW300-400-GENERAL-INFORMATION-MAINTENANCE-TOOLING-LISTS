# CW300/400 General Information, Drawings, Maintenance & Tooling Lists

Scanned engineering documentation for the **Cair Aviation CW300 (C of G / belly) and CW400 (aerotow nose) Ottfur QR release units** — the general information, maintenance, and tooling section of the Cair documentation set. The cover sheet is printed "Part Three" and hand-corrected in red to "Part Five"; the package carries red "Def 1" stamps throughout, consistent with the Definition 1 (April 2009) state recorded in the CW300 amendment record.

## ⚠️ Critical notes for refurbishment work

1. **Sub-assemblies MUST NOT be exchanged between units, and the adjuster must not be re-set or tampered with during a spring change** — this preserves the ISU Datum (CW400SCP Important Note). If the unit fails the Log Book tests after a spring change, it goes back for overhaul, not adjustment.
2. **Cross-variant values differ.** CW300 ISU is 7.5 daN (16.5 lbsf); CW400 ISU is 6.9 daN (15.5 lbsf). See the cross-variant table in the verified PDT transcriptions before trusting any remembered figure.
3. CW400SCP §3.1 directs testing to "Para. 4, and if necessary, Para. 5 **in the unit Log Book**" — the paragraph references are to the Log Book, not to this procedure's own sections.
4. 2BA countersunk stop bolt steps apply to the **CW417 lever only**.
